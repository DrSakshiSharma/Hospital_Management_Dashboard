This project is an interactive Hospital Management Dashboard build using PowerBI to monitor and analyze key hospital performance metrics. 
The dashboard helps in understanding patient trends, doctor distribution, revenue status, and operational efficiency for better healthcare decision-making.

The dashboard provides a complete overview of hospital operations including:
Total Patients: 10K+ patient records analyzed
Average Patient Age: 46 years
Total Doctors: 9,465
Total Revenue: 1.03 Billion
Amount Paid vs Pending: Financial tracking of received and pending payments
Doctor Distribution: Specialization-wise doctor count
Payment Mode Analysis: Revenue by different payment methods
Disease Trend Analysis: Patient count by disease over time
State-wise Financial Overview: Total amount, received and pending by state
Tools & Technologies Used:
Power BI (Dashboard & Visualization)
Microsoft Excel (Dataset)
Data Cleaning & Transformation
DAX for calculated measures
Interactive filters and slicers

Features:
Dynamic and interactive dashboard
Department & specialization insights
Financial and billing analysis
Patient and disease trend visualization
State and city-level filtering
